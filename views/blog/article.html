安装好opensuse42.2之后，默认是KDE桌面，英语环境，键盘布局也是US,这时候是没有中文输入法的。在opensuse下比较好用的输入法是Fcitx。 安装命令： sudo zypper install fcitx 　　这里安装的只是输入法框架，框架内置多种中文输入法，可以选择Google Pinyin或者Pinyin 　　安装命令是： sudo zypper install fcitx-googlepinyin fcitx-pinyin 　　然后修改配置文件应用修改： sudo vi /etc/sysconfig/language 　　将INPUT_METHOD="" 改为 INPUT_METHOD="fcitx" 　　最后修改环境变量： vi ~/.bashrc 　　添加 export GTK_IM_MODULE=fcitx export XMODIFIERS=@im=fcitx export QT_IM_MODULE=fcitx 　　刷新环境变量 source ~/.bashrc 　　此时可以通过右下角的小键盘,打开Configure选项进行设置,切换的快捷键为Ctrl+space 866039660.png 　　在安装过程中发现若没有修改环境变量则无法进行切换。
